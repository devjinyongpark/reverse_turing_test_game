# Installation

1. download the project.
2. add `.env` file.
3. write openai api key in the `.env` file.
   
   ex. `OPENAI_API_KEY = 'sk-...'`

!Note You need openai api key to run the program. You can get it by subscribing opai plus or pro plan.

4. Install python(version 3.11+) from python.org
5. Run Terminal(MacOS) or Command Prompt(Windows).
6. Open the directory where the project downloaded in the terminal/Command Prompt.

   `cd <your-directory>`

8. Type `python main.py`

  If it doesn't work, use `python3 main.py`
